## Air-Quality-Insight-for-Southeast-Asia
Designed and developed Air Quality Insights for Southeast Asia, a web platform focused on visualizing air quality data and raising awareness across Malaysia and Southeast Asia. The platform features a clean, responsive interface with interactive elements and real-time updates to engage users and promote environmental awareness. Download all files, make sure everything is in one folder, and don't forget to unzip the img.zip folder to view the web,!
